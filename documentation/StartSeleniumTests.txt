Per far partire i test di Selenium:
-andare in /var/lib/jenkins/workspace/selenium_test
-startare il server xvfb con "Xvfb :0 &"
-salvare il display con "export DISPLAY=:0"
-eseguire il file java con il main
-se non funziona provare a killare i processi con:
	-pkill selenium
	-pkill geckodriver
	-pkill firefox
